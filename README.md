### 1. Persiapan File dan Folder
Buat folder bernama `Lab4Web` dan file pertama `lab4_box.html`.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mengenal JavaScript</title>
</head>
<body>
      <h1>Pengenalan JavaScript</h1>
      <3>Contoh document.write dan console.log</h3>
      <script>
        document.write("Hello World");
        console.log("Hello World");
      <script>
</body>
</html>
```

### 4. Membuat Layout Sederhana

Buat file baru bernama `home.html` dan `style.css` di dalam folder `lab4_layout`.

Struktur HTML:

``` <!DOCTYPE html> 
    <html lang="en"> 
    <head>
      <meta charset="UTF-8"> 
      <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
      <title>Layout Sederhana</title> 
      <link rel="stylesheet" href="style.css"> 
    </head> 
    <body> 
      <div id="container"> 

      </div> 
    </body> 
    </html>
```
